This mouse_volume.ahk file allows you to change volume using your mouse.

1. Run `test_button.ahk` and tap desired side button

      Remember name
   
      To stop script, right click hot key icon in hidden tray and hit exit/pause

3. Edit `mouse_volume.ahk`
   
      Edit `SideBtn := "XButton2"` to your side button

4. Execute

      `./mouse_volume.ahk` or double click file
